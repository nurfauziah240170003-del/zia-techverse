## 📖 Tentang Proyek

Zia TechVerse adalah website blog berbasis Laravel yang membahas teknologi, Artificial Intelligence (AI), dan inovasi digital.

Website ini dibuat sebagai proyek mata kuliah Pemrograman Web.

---

## ✨ Fitur Utama

- Login dan Register
- Dashboard Admin
- CRUD Artikel
- CRUD Kategori
- Upload Thumbnail
- Sistem Komentar
- Pencarian Artikel
- Filter Kategori
- Tampilan Responsif
